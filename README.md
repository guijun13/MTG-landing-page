## Project Setup

### Requirements

1. node.js 18+
2. npm 8+

### Clone the project

```sh
git clone https://github.com/guijun13/MTG-landing-page.git
```

### Install dependencies

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
